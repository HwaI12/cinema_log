これは[`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app) を使用して起動した [Next.js](https://nextjs.org/) プロジェクトです。

## はじめに

まず、開発サーバーを実行します：

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて結果を確認してください。
